¿Dónde buscar datos de Biodiversidad?

Portal de datos de GBIF

  • Ir al portal de datos de http://datos.gbif.es

  • Realiza una búsqueda por el nombre de un taxon, e.g. Quercus pyrenaica, en la barra de búsqueda y clica en el botón de búsqueda

GBIF API. Ejemplo

https://api.gbif.org/v1/species/match?name=Quercus pyrenaica

  • URL base: https://api.gbif.org/v1
  • Grupo/Namespace que queremos consultar: species, occurrences, etc. Mas información en documentación de Referencia
  • Función: funcionalidad a usar: buscar (search), coincidencia (match), etc.
  • Parámetro: los parámetros para nuestra llamada: e.g. name.
  • consulta: la consulta dentro de dicho parámetro
